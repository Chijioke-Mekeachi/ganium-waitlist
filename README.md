# Ganium Waitlist Web

Dark-mode Next.js waitlist app with a small admin page.

## Setup

1) Install

```bash
npm install
```

2) Env

Copy `waitlist-web/.env.example` to `waitlist-web/.env` and fill in values.

3) Supabase

Run `waitlist-web/docs/waitlist.sql` in the Supabase SQL editor.
If you already ran an older version, run `waitlist-web/docs/waitlist_migration.sql` instead.

4) Run

```bash
npm run dev
```

## Routes

- `/` waitlist
- `/thanks` success
- `/admin` admin login + list + mark contacted
